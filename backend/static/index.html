SWEThePeople API

Search:

/search/?query=<query>
Searches the database for any matches to the given query

Representative Endpoints:

/representative
Returns list of all the U.S. representatives

/representative/<bioguide>
Returns information about representative based on his or her bioguide id

Party Endpoints:

/party
Returns list of all U.S. Political Parties.

/party/<name>
Returns information about a political party based on its name.

State Endpoints:

/state
Returns list of all U.S. states

/state/<USPS state abbreviation>
Returns a single state

District Endpoints:

/district
Returns list of all U.S. congressional districts

/district/<USPS state abbreviation>
Returns list of all U.S. congressional districts in a state

/district/<USPS state abbreviation>/<district id>
Returns a single congressional district
